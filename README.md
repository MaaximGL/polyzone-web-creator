# polyzone-web-creator

Минималистичный web-инструмент для создания и отладки зон на карте GTA V/FiveM.

## Что это

`polyzone-web-creator` сделан для личной рабочей задачи: быстро создавать зоны, проверять их на карте и экспортировать результат в удобном виде.

Проект создан на базе ресурса `trgl_debughud`, как web-часть.

###### `trgl_debughud` - универсальный ресурс для runtime работы с окружением внутри игры.

## Возможности

- Создание зон:
  - `Poly` (free)
  - `Circle`
  - `Rectangle`
- Редактирование, удаление и переименование зон
- Экспорт и импорт:
  - `JSON`
  - `LUA`
- Отдельные tiles для:
  - `Los Santos`
  - `Cayo Perico`
- Grid / Overlay / Tile Debug для проверки тайлов и координат
- Отображение координат курсора на карте

## Назначение

Проект не претендует на универсальный публичный редактор.  
Это прикладной инструмент под конкретный workflow для быстрой работы с зонами.
